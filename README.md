## 🚀 Projects
#### ⭐ Featured Open Source

### 🏥 MediCheck
Bayesian ML-powered symptom checker with real-time health analysis and robust containerized deployment. Features robust FastAPI endpoints, Streamlit dashboard interface, hardened non-root multi-stage Docker builds, and continuous integration/deployment pipeline via GitHub Actions. Translates uncertain symptoms into explainable, probabilistic triage signals with a strong operational bias.
- **Tech Stack:** `Python`, `FastAPI`, `Streamlit`, `Scikit-Learn`, `Docker`, `GitHub Actions`
- **Source Code:** [MohitSongra/medicheck 🛠️](https://github.com/MohitSongra/medicheck)

### 🔍 PixelSearch
AI-powered semantic photo retrieval engine using vision-language models for natural language image search. Integrates CLIP embeddings, FAISS indexing, and BLIP-2 image captioning alongside experimental FaceNet-based facial recognition. Transforms unstructured image libraries into a highly searchable, context-aware semantic memory pipeline.
- **Tech Stack:** `Python`, `CLIP`, `FAISS`, `BLIP-2`, `FaceNet`, `PyTorch`
- **Source Code:** [MohitSongra/PixelSearch 🛠️](https://github.com/MohitSongra/PixelSearch)

### 🏆 AI Performance Marketing SaaS
**Hackbuild 2.0 Winning Platform 🏆** built for AI-driven ad campaign optimization, predictive performance, and ROI forecasting. Incorporates real-time analytics dashboards, audience targeting engines, and Llama 3/Groq-powered targeting insights to turn complex campaign metrics into guided marketing strategies.
- **Tech Stack:** `JavaScript`, `React`, `Node.js`, `Groq API`, `Llama 3`
- **Source Code:** [GDGVITM/hackbuild-Invictus 🛠️](https://github.com/GDGVITM/hackbuild-Invictus)

### 🌱 EcoQuest
Gamified e-waste recycling and tracking platform promoting sustainable waste management behavior. Integrates real-time drop-off location tracking, Firebase-backed user activity streams, rewards loop mechanics, environmental impact metrics dashboards, and community-wide leaderboard mechanics.
- **Tech Stack:** `Next.js`, `TypeScript`, `Firebase`, `TailwindCSS`, `Vercel`
- **Source Code:** [MohitSongra/EcoQuest 🛠️](https://github.com/MohitSongra/EcoQuest)
